# ggml-backend (planned — the foundational enabler)

An XDNA NPU backend for ggml/llama.cpp so the NPU can participate in LLM inference directly
(not just as a coprocessor). **Status: not built — the biggest lift, biggest payoff.** Until then,
the NPU's measured LLM role is the prune/vision coprocessor (see open-xdna).
